# Contributing

Thanks for your interest in `bim-revit-utilities`! This is a personal academic project, but contributions are very welcome.

## How to contribute

1. **Open an issue first** for any non-trivial change so we can discuss the design.
2. Fork the repo and create a feature branch: `git checkout -b feature/my-improvement`.
3. Follow the existing code style (PEP 8, type hints where practical).
4. Add or update tests / examples for new functions.
5. Open a Pull Request with a clear description and link to the issue.

## Scope

This repo focuses on:

- Parsing Revit schedule exports (CSV, Excel)
- Quantity take-off helpers (walls, slabs, openings)
- Cost-group mapping (DIN 276 / HOAI workflows)
- German construction-standard helpers (ZTV-ING, VOB)

Out of scope: full Revit plug-ins, proprietary IFC parsers, GUI tools.

## Code of Conduct

Be respectful and constructive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/) v2.1.
